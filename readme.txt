WordPress Sitenizde Facebooktaki “Ne Düşünüyorsunuz” Bölümü İle Twitter Karışımı Bir Sistemdir.

== Description ==
**Türkçe**

**Eklenti Ne İşe Yarar?**

WordPress Sitenizde Facebooktaki “Ne Düşünüyorsunuz” Bölümü İle Twitter Karışımı Bir Sistemdir.

Kısa İletilerinize Görseller Ekleyerek Bloğunuzda / Sitenizde Yayınlamanızı Sağlar.

**Nasıl Kullanılır?**

Admin Panelinizde Başlangıç (Dashboard) Sayfasında Oluşturulan Sekme Sayesinde İleti Girebilirsiniz.

Panelin Hakkında Bölümüne Tıklayarak Shortcodeları Görebilirsiniz.

**SHORTCODELAR**

İletilerinizi Göstermek İstediğiniz Sayfada / Yazı İçerisinde Kullanmanız Yeterli Olucaktır.

Eklenti İle Gelen 2 Tür Shortcode Vardır.

[alixcan_live_feed] – Tüm İletileri Listeler

[alixcan_live_feed id=""] – Belirlediğiniz İletiyi İstediğiniz Yerde Listeler





**English**

**What For?**

This Is a Mixture Of Twitter And Facebook (What Do You Think) Sections.

Its Allowed To Send A Short Message And Image On Your WordPress Site Like Facebook Or Twitter.

**How To Use?**

Go To Your Admin Panel Dashboard And Start To Use

**Shortcodes**

There are two short code.

First one is `[alixcan_live_feed]` - its list all feeds in your post or page.

Second shordcode is `[alixcan_live_feed id=""]` - its get the one feed whitch id you want

== Installation ==

**Türkçe**

1. `alixcan_live_f` Klasörünü `/wp-content/plugins/` İçerisine Yükleyin.
1. Eklentiler Sayfasından Eklentiyi Aktif Edin
1. Başlangıç Sayfasından Kullanmaya Başlayabilirsiniz.

## Documentation
Daha Fazla Yardıma İhtiyaç Duyarsanız [Tıklayın.](http://www.alixcan.net/?p=1200). 





**English**

1. Upload `alixcan_live_f` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin in the 'Options' (or 'Settings') screen by following the onscreen prompts.

## Documentation
If you need more help installing and configuring the plugin, [see here for more information](http://www.alixcan.net/?p=1200).
